# Chaos Delivery

**Chaos Delivery** — это игра, в которой ты в лице главного героя пытаешься бороться со скукой. К счастью тебе встречается странный мальчишка, у которого странным образом появился свой сервис доставки. 
Твоя задача — доставить все заказы вовремя, а также не задавать вопросы по поводу содержимого груза.
